# Code
Based on what you wish to log, you can determine the importance which influences the visual response.

```cs
Log.Info( "This is an Info" );
Log.Warning( "This is a Warning" );
Log.Error( "This is an Error" );
```

## Status
To gain a basic overview, how much Informational, Warnings or Errors have been thrown in total.
You can look in the bottom right to gain an overview and filter.

Each importance-level can be individually toggled, all of them are active by default
